# 프리코스 2주차 미션 - 자동차 경주

## 구현할 기능 목록

### [입출력 기능]
- 입력
  - [ ] 사용자에게 자동차 이름 입력받기
  - [ ] 사용자에게 시도할 횟수 입력받기
- 출력
  - [ ] 자동차 이름 입력 문구 출력하기
  - [ ] 시도할 횟수 입력 문구 출력하기
  - [ ] 각 차수별 실행 결과 출력하기
  - [ ] 우승자 안내 문구 출력하기

### [게임 기능]
- 전진 또는 멈춤
  - [ ] 무작위 값을 가져와 4 이상이면 전진
- 우승자가 여러 명일 경우
  - [ ] 쉼표(,)를 이용하여 구분

### [예외 처리]
- 사용자에게 자동차 이름 입력받을 때
  - [ ] 공백은 허용하지 않음
  - [ ] 최소 두 개의 자동차가 있어야 함
  - [ ] 자동차 이름은 쉼표(,)를 기준으로 구분해야 함
  - [ ] 자동차 이름은 1자 이상, 5자 이하만 가능함
- 경기 횟수 입력받을 때
  - [ ] 공백은 허용하지 않음
  - [ ] 정수만 입력 가능함 (소수점, 음수 등 X)
  - [ ] 1이상만 허용함
